# Contributing to Learning SQL

This is a personal learning repository, but feedback and suggestions are welcome!

## How to Contribute

### Reporting Issues

If you spot any errors or have suggestions:

- Open an issue describing the problem
- Include specific file references if applicable
- Suggest corrections or improvements

### Suggesting Resources

If you know of useful SQL resources:

- Open an issue with the resource link
- Explain why it's helpful
- Specify which section it relates to

## Style Guidelines

- Use clear, concise language
- Follow existing markdown formatting
- Include code examples where helpful
- Add source attribution for external content

## Code Examples

- Test all SQL examples before submitting
- Include comments for complex queries
- Specify which database system (SQLite, PostgreSQL, etc.)
- Show expected output when relevant

---

Thank you for helping improve this learning resource!
